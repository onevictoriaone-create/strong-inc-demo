STRONG INC. 24/7 GYM — LUXURY MULTIPAGE WEBSITE

This is a complete static website ready for upload to GitHub Pages, Netlify, Vercel, or any standard web host.

FILES INCLUDED:
- index.html
- about.html
- memberships.html
- gallery.html
- contact.html
- style.css
- script.js
- assets/ folder with logo and gym photos

HOW TO UPLOAD TO GITHUB PAGES:
1. Create a new GitHub repository.
2. Upload every file in this folder, including the assets folder.
3. Go to Settings > Pages.
4. Choose Deploy from branch.
5. Select main branch and root folder.
6. Save and wait for GitHub to publish the website.

CONTACT FORM NOTE:
The contact form is static-host friendly. It opens an email draft to Stronginc24@gmail.com with the visitor's information.
